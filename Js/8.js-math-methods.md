
8. **Math.sqrt(x)** â†’ Returns square root of `x`.

   ```js

   ```



11. **Math.sin(x), Math.cos(x), Math.tan(x)** â†’ Trigonometry functions (radians).


