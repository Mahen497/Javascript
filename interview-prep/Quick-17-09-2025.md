## JavaScript Objects

## **all commonly used JavaScript Number methods** — both **instance methods** (used on number values) and **static methods** (used on `Number.` class) — with **short examples**.

---

## ⚙️ Instance Methods

(Use them on number values like `let n = 123.45;`)

---

### 🧮 `toExponential()`

Shows number in **exponential (scientific)** form.

```javascript
let n = 1234.56;
console.log(n.toExponential()); // "1.23456e+3"
```

---

### 💰 `toFixed(digits)`

Formats number with **fixed decimals**.

```javascript
let n = 5.6789;
console.log(n.toFixed(2)); // "5.68"
```

---

### 📍 `toPrecision(digits)`

Shows number with **total number of digits**.

```javascript
let n = 123.456;
console.log(n.toPrecision(4)); // "123.5"
```

---

### 🌍 `toLocaleString()`

Formats number according to **local settings**.

```javascript
let n = 1234567.89;
console.log(n.toLocaleString()); // e.g. "1,234,567.89"

console.log(num.toLocaleString("en-US")); // US format
console.log(num.toLocaleString("en-IN")); // Indian format

```

---

### 🔡 `toString()`

Converts number to **string**.

```javascript
let n = 255;
console.log(n.toString());    // "255"
console.log(n.toString(16));  // "ff" (hex)
```

---

### 🧠 `valueOf()`

Returns the **primitive number value** of a Number object.

```javascript
let n = new Number(10);
console.log(n.valueOf()); // 10
```

---

## ⚙️ Static Methods

(Use them on `Number.` class like `Number.isNaN(...)`)

---

### ⚠️ `Number.isNaN(x)`

Checks if value is **NaN** (not a number).

```javascript
console.log(Number.isNaN("abc"));    // false
console.log(Number.isNaN(NaN));      // true
```

---

### ♾️ `Number.isFinite(x)`

Checks if number is **finite**.

```javascript
console.log(Number.isFinite(100));   // true
console.log(Number.isFinite(Infinity)); // false
```

---

### 🔢 `Number.isInteger(x)`

Checks if number is an **integer**.

```javascript
console.log(Number.isInteger(10));     // true
console.log(Number.isInteger(10.5));   // false
```

---

### 🛡️ `Number.isSafeInteger(x)`

Checks if number is a **safe integer**.

```javascript
console.log(Number.isSafeInteger(10));            // true
console.log(Number.isSafeInteger(9007199254740992)); // false
```

---

### 📥 `Number.parseInt(str)`

Converts **string → integer**.

```javascript
console.log(Number.parseInt("25px")); // 25
```

---

### 📥 `Number.parseFloat(str)`

Converts **string → float**.

```javascript
console.log(Number.parseFloat("25.7px")); // 25.7
```

# JavaScript `substring()` Method Examples

The `substring(start, end)` method returns the part of the string between the specified start and end indexes.

- `start` → starting index (inclusive)
- `end` → ending index (exclusive)

If:
- `start > end`, the values are **swapped**.
- `start` or `end` is **negative**, it's treated as `0`.

---

## 🧩 Example Code

```javascript
let text = "JavaScript";

// Extract from index 0 up to (but not including) index 4
let result1 = text.substring(0, 4); 
console.log(result1); 
// Output: "Java"

// Extract from index 4 to the end of the string
let result2 = text.substring(4); 
console.log(result2); 
// Output: "Script"

// startIndex is greater than endIndex, so they are effectively swapped
let result3 = text.substring(7, 2); 
console.log(result3); 
// Output: "vaScr"

// Using negative index (treated as 0)
let result4 = text.substring(-5, 3); 
console.log(result4); 
// Output: "Jav"
````

---

✅ **Key Notes**

* Negative values are treated as `0`.
* If `start > end`, JavaScript automatically swaps them.

## 🟢 JavaScript Set — Quick Summary

### 1️⃣ What is a Set?

* Collection of **unique values**.
* Can store **any type**: number, string, object.
* **No duplicates allowed**.

---

### 2️⃣ Creating a Set

```js
let mySet = new Set([1, 2, 2, 3]); 
console.log(mySet); // Set {1, 2, 3}
```

---

### 3️⃣ Common Methods & Properties

| Method / Prop   | Description         |
| --------------- | ------------------- |
| `add(value)`    | Add element         |
| `delete(value)` | Remove element      |
| `has(value)`    | Check if exists     |
| `clear()`       | Remove all elements |
| `size`          | Number of elements  |

---

### 4️⃣ Iteration

```js
for (let v of mySet) console.log(v);
mySet.forEach(v => console.log(v));
```

---

### 5️⃣ Converting Array ↔ Set

```js
let arr = [1,2,2,3];
let unique = [...new Set(arr)]; // [1,2,3]
```

---

### 6️⃣ Common Operations

```js
let a = new Set([1,2,3]);
let b = new Set([2,3,4]);

// Union
let union = new Set([...a, ...b]); // {1,2,3,4}

// Intersection
let intersection = new Set([...a].filter(x => b.has(x))); // {2,3}

// Difference
let difference = new Set([...a].filter(x => !b.has(x))); // {1}
```

---

### ✅ Key Points

* Stores **unique values only**
* Iterates in **insertion order**
* Fast **lookup** with `has()`
* Use for **deduplication** and **set operations**
